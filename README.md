CRAC 业余无线电考试助手 (免登录极速版)

本项目是一个用于监控 CRAC (中国无线电协会业余无线电分会) 官网考试信息并自动推送通知的工具，旨在帮助无线电爱好者第一时间获取考试报名信息。

项目提供 **图形界面 (GUI) 版** 和 **命令行 (CLI) 脚本版** 两个版本，以适应不同用户的桌面或服务器自动化部署需求。全新版本已彻底移除账号登录限制，使用更安全、更极速。

查询频率： 1~2 小时内随机触发一次查询，模拟合理操作频率。

## ✨ 功能特点

- **完全免登录**：直连公开数据接口，无需提供个人真实手机号及密码，0 隐私泄露、0 封号风险。
- **GUI 图形界面支持**：提供简洁直观的配置界面，一键启动监控。
- **多省市并行监控**：支持同时监控多个省份或指定城市、指定类别（A类/B类）的考试计划。
- **三重预警冗余**：
  - **桌面弹窗与声音**：第一时间在电脑屏幕弹出提示，并伴随系统提示音（GUI版专用）。
  - **企业微信推送**：支持企微自建应用进行跨设备多端实时消息推送。
  - **本地存根记录**：自动生成 `.txt` 历史发现记录文件，永不丢失。
- **智能化后台运行**：
  - **开机自启**：支持 Windows 注册表开机静默启动后台运行。
  - **系统托盘隐藏**：关闭主界面窗口可自动缩至系统托盘，不占用任务栏。

## 📖 简易使用教程

本项目包含两种主要运行方式，请根据您的实际使用场景选择：

### 方式一：使用 GUI 图形界面版 (推荐桌面电脑用户)

这是最简单、直观的使用方式，适合在 Windows 办公电脑或个人电脑上挂机运行。

#### 1. 环境准备
确保您的电脑已安装 Python 3（建议 3.8 及以上）。
打开终端或命令提示符，安装运行所需的必要依赖库：
```bash
pip install requests urllib3 pystray Pillow
```

#### 2. 运行程序
在项目根目录运行以下命令开启图形化界面：
```bash
python crac_gui.py
```

#### 3. 参数配置与启动
1. **选择目标**：配置 **监控省份**（例如：`福建,浙江`，多省以英文逗号分隔）。
   - 可选：填写 **关注城市**（如：`泉州,厦门`），留空则默认全省监控。
   - 可选：选择您关心的考试类别（A类 / B类 / 不限）。
2. **企微推送（可选）**：如果需要手机端实时接收微信消息，请填入企业微信自建应用的 `CorpID`、`Secret` 和 `AgentID`。
3. **自动运行（可选）**：勾选 **“开机自启(关闭隐藏)”**，程序将在电脑每次开机后自动启动并在后台静默运行。
4. **启动监控**：点击 **▶ 启动监控与自动巡检** 按钮即可开始工作。在此工作模式中，您可以随时将窗口关闭，程序将自动缩至右下角系统托盘后台继续静默为您运行。
   - _提示：您可以点击 **📂 历史发现** 按钮打开本地生成的 `.txt` 文件，查看以往监控到的所有考试记录。您的配置参数也会在启动时自动保存到本地。_

---

### 方式二：使用 CLI 命令行脚本版 (适合服务器/软路由自动化部署)

适合部署在 Linux 服务器、OpenWrt 软路由、NAS 环境或需要依赖各类计划任务高级执行的用户。

#### 1. 环境准备
确保已安装 Python 3 和核心网络库：
```bash
pip install requests
```

#### 2. 参数配置
您可以直接使用任何文本编辑器打开 `crac.py`，在其首部的 **[全局配置区]** 修改相关参数；**（推荐）** 或者在同目录下新建一个名为 `config.py` 的文件，进行独立配置以避免更新时冲突：
```python
# config.py 示例

# 企业微信推送设置（可选，不推则留空）
WECHAT_CORP_ID = 'xxx'
WECHAT_SECRET = 'xxx'
WECHAT_AGENT_ID = 'xxx'

# 监控规则
TARGET_PROVINCES = ['福建', '浙江']
TARGET_CITIES = [] # 留空代表全省，填入城市名如 ['泉州', '厦门'] 则只监控特定城市
```

#### 3. 运行与部署安排
您可以直接执行它：
```bash
python crac.py
```
**建议自动化部署方式**：
- **Linux / OpenWrt / Mac**: 使用 `crontab -e` 设置每小时或每半小时执行一次。示例设置：`0 * * * * /usr/bin/python3 /您的实际路径/crac.py`
- **Windows**: 使用系统自带的“任务计划程序”设置定期定点触发执行 `crac.py`。
（注意：`crac.py` 内部脚本自带一定程度的随机防刷机制，请放心部署。）

## 🛠️ 将 GUI 打包为独立 EXE (Windows)

如果您不希望每次都在电脑上配置繁琐的 Python 环境，想将其生成一个可以直接分享并在 Windows 电脑上双击运行的独立程序（`.exe`），请使用 `PyInstaller` 进行打包：

1. 安装打包工具：
```bash
pip install pyinstaller
```
2. 执行打包命令：
```bash
pyinstaller --noconsole --onefile --icon=logo.ico crac_gui.py
```
打包成功后，位于本项目 `dist` 文件夹下的 `crac_gui.exe` 即为您所需要的可执行独立文件。
